# Flipkart Automation Assignment 🛒

This project is a **Selenium WebDriver automation script** written in Java.
It automates a product search and purchase flow on Flipkart.

## 📌 Features

* Opens Flipkart website
* Searches for **Bluetooth Speakers**
* Applies filters:

  * Brand: boAt
  * Rating: 4★ & above
  * Price filter
* Selects a product
* Switches to new window
* Adds product to cart
* Navigates to cart page
* Takes screenshot of cart

## 🛠️ Tech Stack

* Java
* Selenium WebDriver
* ChromeDriver

## ▶️ How to Run

1. Install Java & set environment variables
2. Add Selenium dependencies (JAR)
3. Download ChromeDriver and set path
4. Run the `Assignment.java` file
